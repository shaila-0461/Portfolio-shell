# Landing Page Project

##  Project Description
This project is a simple and responsive **Landing Page** created using **semantic HTML** and basic **CSS**.  
It demonstrates the proper use of HTML5 semantic elements and includes a **working contact form** with name, email, and message fields.

The project is designed for learning and academic purposes, focusing on clean structure, readability, and best practices in front-end development.

---

## Project Purpose
The purpose of this project is to:
- Practice semantic HTML structure
- Understand the use of HTML5 elements like header, nav, main, section, article, and footer
- Create a functional contact form
- Build a basic landing page layout suitable for beginners

---

##  Technologies Used
- **HTML5** – for page structure and semantic elements    
- **Web Browser** – to view and test the project  

---

## ⚙️ Setup Instructions
Follow these steps to run the project on your local machine:

1. Download or clone the project files.
2. Make sure the file is named **`index.html`**.
3. Place the file in any folder on your computer.
4. Open the file by double-clicking it  
   **OR**
   Right-click → Open with → Any web browser (Chrome, Edge, Firefox).
5. The landing page will open and display correctly in the browser.
